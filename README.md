### A Simple Search Page

* 可以搜索M平台上的用户，品牌，电商，关键词，用于练习和试验新前端技术
* run the project
    * clone the project
        ```
        git clone https://github.com/wangdejun/SimpleReactSearchExample.git
        ```
    * install the dependencies
        ```
        npm install
        ```
    * start and test
        ```
        webpack-dev-server
        ```
